# Emmet

### 生成HTML骨架

```apl
!加TAB
```

### 注释

```apl
ctrl + /
```

### 生成父子关系标签

```apl
ul>li加TAB
div>p加TAB
```

### 生成兄弟关系标签

```apl
div+p加TAB
```

### 生成父级关系标签

```apl
div>p^p
```

### 分组

```apl
div(div>p*2)
```

### 乘法

```apl
p*10加TAB
```

### 自增

```apl
div.$*5
```

### ID

```apl
#name
```

### 类

```apl
.name
```

### 文本

```apl
div{kkk}
```
