# Maven 

```xml
<groupld>Maven 项目隶属组织名称</groupld>
<artifctld>定义 Maven 项目名称</artifctld>
<version>定义当前项目版本号</version>
```

````xml
<!--导入依赖包-->
    <dependencies> 
        <dependency>
            <groupId></groupId>
            <artifactId></artifactId>
            <version></version>
        </dependency>
    </dependencies>
````
